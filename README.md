# Hostel Management System

A traditional Java EE Hostel Management System developed using **Java, JSP, Servlets, JDBC, and MySQL**. The application provides separate **Admin** and **Student** portals to simplify hostel operations such as room allocation, student registration, complaint handling, and leave management.

---

## Project Overview

This project was originally developed as a **BCA Final-Year Academic Project** using traditional Java EE technologies. It has been enhanced with an improved user interface using the **Antigravity AI Coding Agent**, while preserving the original **JSP + Servlet + JDBC** architecture and beginner-friendly coding style.

---

## Features

### Admin Module

- Secure Admin Login
- Dashboard
- Change Password
- Room Management
  - Add Room
  - Update Room
  - Delete Room
  - View Rooms
- Hostel Registration Management
- Student Management
- Complaint Management
- Leave Management

### Student Module

- Student Registration
- Student Login
- Profile Management
- Change Password
- View Available Rooms
- Hostel Registration & Room Booking
- View Booking Details
- Register Complaints
- Track Complaint Status
- Apply Leave
- View Leave Status
