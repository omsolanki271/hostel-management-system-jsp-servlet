# Hostel Management System

A web-based application designed to manage day-to-day hostel operations, including room booking, student registration, complaints, and leave applications. It facilitates smooth communication and management between the hostel administration and residents.

## Project Description

Hostel Management System developed using Java, JSP, Servlets, JDBC, and MySQL. The application includes Admin and Student modules with features such as room booking, hostel registration, complaint management, and leave management. Originally developed as a BCA final-year academic project and later enhanced with an improved user interface using the Antigravity AI coding agent while preserving the traditional JSP + Servlet + JDBC architecture.

## Features

### Admin
